Script trình bày (copy vào Proposal)

Em đại diện nhóm xin trình bày đề tài nhóm:

1. Mô tả dự án
Hệ thống Movie Ticket Booking System được xây dựng trên Amazon Web Services (AWS) theo kiến trúc Serverless kết hợp Event-Driven Architecture. Hệ thống hỗ trợ người dùng đặt vé, thanh toán trực tuyến, nhận vé điện tử qua Email và Check-in bằng QR Code. Mục tiêu: khả năng mở rộng, tính sẵn sàng, bảo mật và tối ưu chi phí vận hành.

2. Mô tả mô hình
Route 53, CloudFront, WAF và S3 phân phối giao diện và bảo vệ ứng dụng. Amazon Cognito quản lý đăng nhập và phân quyền User/Admin. API Gateway tiếp nhận yêu cầu và chuyển đến các Lambda Service: User, Booking, Payment, Check-in, QR Generator, Notification và Admin. Dữ liệu lưu trên ElastiCache Redis, DynamoDB và RDS MySQL (qua RDS Proxy), kèm AWS Backup. Sau thanh toán thành công, EventBridge và SQS FIFO xử lý bất đồng bộ; QR Code lưu S3 và gửi Email qua SES. CloudWatch + SNS giám sát; CodeCommit, CodePipeline, CodeBuild triển khai CI/CD.

3. Luồng hoạt động
Người dùng truy cập website, đăng nhập Cognito, gửi yêu cầu API Gateway. Lambda xử lý đặt vé, giữ ghế, thanh toán và lưu dữ liệu. Thanh toán xong → EventBridge/SQS → tạo QR → S3 → SES gửi vé. Tại rạp, Staff quét QR qua camera, hệ thống xác thực và cập nhật Check-in. CloudWatch theo dõi metrics/logs; Alarm + SNS cảnh báo Developer; Lambda deploy qua CI/CD.

Luồng đánh số (theo sơ đồ nhóm)

  1. 1 User/Admin/Staff truy cập HTTPS qua Route 53 → CloudFront → WAF → S3 (SPA).
  2. 2 Đăng nhập Cognito (User hoặc Admin pool) → nhận JWT.
  3. 3 Frontend gọi API Gateway kèm JWT.
  4. 4 API Gateway invoke Lambda Service tương ứng.
  5. 5 Lambda đọc/ghi Data Layer (Redis, DynamoDB, RDS qua Proxy).
  6. 7 Sau booking/payment thành công → publish event EventBridge.
  7. 8 EventBridge route → SQS FIFO (lỗi → DLQ).
  8. 9 Payment Lambda redirect sang Payment Gateway (ngoài AWS).
  9. 10a IPN/Webhook callback về API Gateway · 10b User quay lại web.
  10. 11 QR Generator Lambda tạo QR, lưu S3.
  11. 12–13 Notification Lambda lấy QR từ S3 → SES gửi email vé.
  12. 14–15 Staff scan QR → Check-in Lambda xác thực → trả kết quả.
  13. 16 CodePipeline deploy Lambda từ CodeCommit.
  14. 17–18 CloudWatch logs/metrics → Alarm → SNS → Developer.
